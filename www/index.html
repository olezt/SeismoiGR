SeismoiGR

Earthquakes live


This application is a hybrid application that is designed to illustrate the activity of earthquakes worldwide in real time. Technologies used are Javascript and HTML5, Apache cordova, Ionic and AngularJS frameworks.
Currently supports Greek and English.

Data are retrieved from SeismicPortal only for educational purposes.
